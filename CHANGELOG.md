## [0.1.0](https://github.com/bigknoxy/joshify/compare/0.0.0...v0.1.0) (2026-03-31)


### Features

* Initial release of Joshify TUI Spotify client
* OAuth 2.0 authentication with credential caching
* Playback controls (play/pause, skip, seek, volume)
* Browse playlists and liked songs
* Search for tracks
* Album art fetching with two-level caching (memory + disk)
* Focus-based TUI navigation
* Keyboard shortcuts for all actions
* Queue management (view and add tracks)
* npm/bun installation support
* One-line installer and uninstaller scripts
